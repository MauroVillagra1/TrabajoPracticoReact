# Web del clima

Tabla de Contenidos<br>
Web del clima - Dificultad: 🟢🟡🔴<br>
Crea una aplicación web, que permita escribir una ubicación y un país, luego presione un botón consultar y muestre el clima del lugar ingresado, consultar el clima utilizando la API https://openweathermap.org/. Si la ubicación ingresada no existe, mostrar un cartel de error indicando que no se encontraron datos de la ciudad ingresada.<br>
Tecnologias usadas 👩🏻‍💻<br>
HTML5 CSS React-Bootstrap Javascript React<br>
